Data Sources

Uses OpenStreetMap Nominatim for geocoding, OSRM for route distance and base duration, Overpass API for mapped red-light counts, and Open-Meteo Air Quality for route context. Live traffic and true historical traffic are estimated with a local Delhi NCR time-of-day model because free public APIs rarely expose reliable live or historical congestion data without keys.

Useful next additions: metro-first routing, EV/CNG support, toll and parking estimates, user-specific mileage history, and a backend cache to respect public API rate limits.
